---

## Repository Structure

Projects are organized into individual folders under the `projects/` directory.  
Each project folder contains:

- **One Excel workbook** including:
  - Dataset(s)
  - Data cleaning and transformation
  - Analytical calculations and modeling
  - Charts and/or interactive dashboards
  - An executive summary sheet
- **A dashboard or model preview image** for quick visual context
- **A project-specific README** explaining objectives, methods, and insights

This structure allows reviewers to quickly understand each project without opening the Excel file immediately.

## How to Review a Project

1. Open the project folder of interest
2. Review the **dashboard preview image** for a high-level overview
3. Open the **Excel workbook**
4. Start with the **Executive Summary** or **START_HERE** sheet
5. Explore the underlying data, calculations, and dashboards as needed
