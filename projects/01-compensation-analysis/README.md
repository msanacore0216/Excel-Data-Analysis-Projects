# Compensation Analysis Case (Excel)

## Overview
This project evaluates compensation structures using a single Excel workbook that integrates data, analysis, modeling, and executive-level insights.

## Workbook Contents
The Excel file includes:
- Raw compensation data
- Cleaned and structured datasets
- Analytical calculations and comparisons
- Charts and/or dashboard views
- Executive summary sheet highlighting key insights

## Key Questions
- How do compensation levels compare across roles or groups?
- What trends or disparities are present in the data?

## Tools & Techniques
- Excel formulas and financial functions
- PivotTables
- Charts and visual summaries
- Structured modeling approach

## How to Use
1. Download the Excel file
2. Enable editing when prompted
3. Review the **Executive Summary** and **Dashboard** sheets first
