# Gaming Capital Budgeting Case (Excel)

## Overview
This project evaluates a capital investment decision within the gaming industry using capital budgeting techniques to assess project feasibility.

## Workbook Contents
The Excel file includes:
- Cash flow projections
- Discounted cash flow calculations
- NPV, IRR, and payback analysis
- Sensitivity or scenario analysis (if applicable)
- Executive summary of investment recommendation

## Key Questions
- Is the proposed capital investment financially viable?
- How sensitive are returns to changes in key assumptions?

## Tools & Techniques
- Time value of money calculations
- NPV and IRR modeling
- Excel financial functions
- Structured investment evaluation framework

## How to Use
1. Download the Excel file
2. Enable editing
3. Review cash flow assumptions and capital budgeting outputs
