# Mutual Fund 7-Year Performance Review (Excel)

## Overview
This project analyzes the long-term performance of a mutual fund over a seven-year period, focusing on returns, risk, and consistency.

## Workbook Contents
The Excel file includes:
- Historical return data
- Performance metrics and calculations
- Comparative benchmarks
- Charts illustrating trends and volatility
- Executive summary of performance evaluation

## Key Questions
- How has the mutual fund performed over time?
- How does performance compare to relevant benchmarks?

## Tools & Techniques
- Return and performance calculations
- Risk metrics
- Excel formulas and charts
- Time-series analysis

## How to Use
1. Download and open the Excel file
2. Enable editing
3. Review performance charts and summary conclusions
