# Real Estate Financial Modeling Case (Excel)

## Overview
This project develops a financial model to evaluate a real estate investment, incorporating cash flows, financing, and return metrics.

## Workbook Contents
The Excel file includes:
- Property assumptions and inputs
- Operating cash flow projections
- Financing and return calculations
- Investment performance metrics
- Executive summary of investment viability

## Key Questions
- Is the real estate investment financially attractive?
- What returns can be expected under different assumptions?

## Tools & Techniques
- Financial modeling in Excel
- Cash flow analysis
- Return metrics (e.g., IRR, cash-on-cash)
- Structured real estate evaluation

## How to Use
1. Download the Excel file
2. Enable editing
3. Review assumptions, projections, and return metrics
