# Credit Card Strategy Analysis (Excel)

## Overview
This project evaluates credit card usage strategies to optimize rewards, benefits, and overall financial value.

## Workbook Contents
The Excel file includes:
- Credit card features and rewards data
- Comparative analysis across card options
- Scenario-based strategy evaluation
- Summary tables and charts
- Executive-level recommendations

## Key Questions
- Which credit card strategies maximize rewards and value?
- How do different spending patterns impact outcomes?

## Tools & Techniques
- Comparative financial analysis
- Excel formulas and lookup functions
- Scenario modeling
- Data visualization

## How to Use
1. Download and open the Excel file
2. Enable editing
3. Review strategy comparisons and summary insights
