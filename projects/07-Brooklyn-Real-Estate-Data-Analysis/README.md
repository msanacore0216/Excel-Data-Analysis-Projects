# Brooklyn Real Estate Data Analysis (Excel)

## Overview
This project analyzes real estate data from Brooklyn to identify pricing trends, market patterns, and investment insights.

## Workbook Contents
The Excel file includes:
- Real estate transaction or listing data
- Cleaned and structured datasets
- Analytical calculations and summaries
- Charts highlighting market trends
- Executive summary of key findings

## Key Questions
- What trends are present in the Brooklyn real estate market?
- How do prices and metrics vary across properties or areas?

## Tools & Techniques
- Data cleaning and preparation
- Excel formulas and PivotTables
- Market trend analysis
- Visual analytics

## How to Use
1. Download the Excel file
2. Enable editing
3. Review charts and summary findings
